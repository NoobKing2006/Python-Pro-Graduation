# Projenin Fikri

"Bilimsel veriler, fosil yakıtların küresel sera gazı salımlarının yaklaşık %75’inden, karbondioksit salımlarının ise %90’ından sorumlu olduğunu gösteriyor."

Bu Bilgilerle, Bir ekocalculator* sitesi Yapabiliriz.



>*“Ekocalculator” ya da Türkçe karşılığıyla “Eko Hesapla”, çevresel etkiyi ölçmek için kullanılan bir hesaplama aracıdır. Genellikle karbon ayak izi, enerji tüketimi ve sürdürülebilirlik performansını hesaplamak amacıyla geliştirilmiştir.
